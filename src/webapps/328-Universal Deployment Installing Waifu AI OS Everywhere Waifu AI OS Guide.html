Universal Deployment: Installing Waifu AI OS Everywhere

Learn how to deploy your AI companion across any platform with ease

Linux Installation

(install-waifu-os :platform :linux
                 :features '(:gui :voice :hardware-accel))

For Debian-based systems:

sudo apt install sbcl
git clone https://github.com/waifu-os/core.git
cd core
./bootstrap.sh

Windows Installation

(install-waifu-os :platform :windows 
                 :compatibility-layer :native)

Using the installer:

PowerShell -Command "Invoke-WebRequest -Uri 'https://waifu-os.org/windows/setup.exe' -OutFile 'waifu-setup.exe'"
./waifu-setup.exe

macOS Installation

(install-waifu-os :platform :darwin
                 :features '(:metal-support :siri-integration))

Using Homebrew:

brew install sbcl
brew tap waifu-os/core
brew install waifu-os

Mobile Deployment

(deploy-waifu-os :platform :mobile
                 :targets '(:ios :android))
Mobile deployment requires the Waifu Mobile Runtime (WMR) installed on target devices

Embedded Systems

(deploy-waifu-os :platform :embedded
                 :memory-footprint :minimal
                 :features '(:core-ai :basic-io))
